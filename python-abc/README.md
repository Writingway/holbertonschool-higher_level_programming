# Python – Abstract Classes and Interfaces

## Description

Projet Python consacré aux classes abstraites et aux interfaces à l’aide du module `abc`, afin de définir des comportements communs et imposer des méthodes aux classes dérivées.

## Objectifs

* Comprendre les classes abstraites (`ABC`)
* Utiliser les méthodes abstraites (`@abstractmethod`)
* Structurer une architecture orientée objet propre

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
