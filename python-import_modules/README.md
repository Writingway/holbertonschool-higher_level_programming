# Python – Import & Modules

## Description

Projet Python dédié à l’utilisation des modules et de l’instruction `import` pour organiser et réutiliser du code.

## Objectifs

* Comprendre `import`, `from`, `as`
* Utiliser des modules intégrés
* Créer et importer ses propres modules

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
