# RESTful API

## Description

Projet consacré à la conception et à l’utilisation d’API RESTful, permettant la communication entre applications via le protocole HTTP et l’échange de données (souvent en JSON).

## Objectifs

* Comprendre les principes REST
* Manipuler les méthodes HTTP (GET, POST, PUT, DELETE)
* Structurer des routes et gérer les réponses

## Technologies

* Python 3
* JSON
* HTTP

## Exécution

```bash
python3 app.py
```
