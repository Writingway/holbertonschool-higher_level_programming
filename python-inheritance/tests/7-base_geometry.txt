===========================
How to use 7-base_geometry.py
===========================

This module defines a base geometry class using the `BaseGeometry` class.

Usage example:
==============

Importing the class from the module:
    >>> BaseGeometry = __import__('7-base_geometry').BaseGeometry

Now we can create a BaseGeometry instance:
    >>> bg = BaseGeometry()

The class has two methods:

1. area(self)
=================

This method is not implemented and should raise an Exception:

    >>> bg.area()
    Traceback (most recent call last):
    Exception: area() is not implemented

2. integer_validator(self, name, value)
==========================================

This method validates that `value` is a positive integer. If not, it raises appropriate exceptions:


Passing valid integers should not raise any exceptions:
    >>> bg.integer_validator("my_var", 10)
    >>> bg.integer_validator("another_var", 1)
