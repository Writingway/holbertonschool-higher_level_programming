# Python – Test Driven Development (TDD)

## Description

Projet Python consacré au développement piloté par les tests (TDD), où les tests sont écrits avant le code.

## Objectifs

* Comprendre le principe du TDD
* Écrire des tests unitaires
* Valider le code par des cas de test

## Technologies

* Python 3

## Exécution

```bash
python3 -m doctest ./tests/*
python3 nom_du_fichier.py
```
