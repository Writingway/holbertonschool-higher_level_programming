# Python – Server Side Rendering (SSR)

## Description

Projet Python consacré au rendu côté serveur (Server Side Rendering) pour générer des pages web dynamiques avant de les envoyer au navigateur.

## Objectifs

* Comprendre le SSR
* Utiliser des frameworks Python comme Flask ou Django
* Générer des templates HTML dynamiques
* Gérer les routes et les données côté serveur

## Technologies

* Python 3
* Flask / Django
* HTML5
* CSS3

## Exécution

```bash
python3 app.py
```
