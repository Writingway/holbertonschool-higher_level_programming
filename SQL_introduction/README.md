# SQL – Introduction

## Description

Projet d’introduction au langage SQL et aux bases de données relationnelles.

## Objectifs

- Comprendre les bases des bases de données relationnelles
- Écrire des requêtes SQL simples
- Manipuler des tables avec `SELECT`, `INSERT`, `UPDATE`, `DELETE`

## Technologies

- SQL
- MySQL

## Exécution

```bash
mysql -u root -p < fichier.sql
```
