# Python – If, Else, Loops & Functions

## Description

Projet d’initiation à Python couvrant les bases du langage : conditions (`if/else`), boucles (`for`, `while`) et fonctions.

## Objectifs

* Comprendre la logique conditionnelle
* Maîtriser les boucles
* Créer et utiliser des fonctions
* Écrire un code clair et lisible

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
