# Python – Input / Output

## Description

Projet Python consacré à la gestion des entrées et sorties : lecture et écriture de fichiers, manipulation de texte et sérialisation de données.

## Objectifs

* Lire et écrire dans des fichiers
* Manipuler du texte et du JSON
* Comprendre la persistance des données

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
