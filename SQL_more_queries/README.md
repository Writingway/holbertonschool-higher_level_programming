# SQL – More Queries

## Description

Projet approfondissant les requêtes SQL avancées et la manipulation de bases de données relationnelles.

## Objectifs

- Utiliser les jointures (`JOIN`)
- Manipuler les contraintes (PRIMARY KEY, FOREIGN KEY, UNIQUE)
- Gérer les permissions utilisateurs
- Écrire des requêtes complexes avec sous-requêtes

## Technologies

- SQL
- MySQL

## Exécution

```bash
mysql -u root -p < fichier.sql
```
