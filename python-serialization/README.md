# Python – Serialization

## Description

Projet Python consacré à la sérialisation des données, permettant de convertir des objets Python en formats stockables ou transférables (comme JSON) et inversement.

## Objectifs

* Comprendre la sérialisation et la désérialisation
* Utiliser le format JSON en Python
* Sauvegarder et restaurer des objets

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
