# Python – Everything is Object

## Description

Projet Python explorant le modèle objet du langage, où tout est considéré comme un objet (variables, fonctions, classes, etc.).

## Objectifs

* Comprendre le concept "everything is object"
* Explorer les types et leur comportement
* Analyser les références et la mutabilité
* Comprendre l’identité (`id`) et le type (`type`)

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
