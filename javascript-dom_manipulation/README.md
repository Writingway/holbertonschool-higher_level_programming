# JavaScript – DOM Manipulation

## Description

Projet JavaScript consacré à la manipulation du DOM pour créer des interactions dynamiques dans une page web.

## Objectifs

* Comprendre le DOM (Document Object Model)
* Sélectionner et modifier des éléments HTML
* Gérer les événements (click, input, etc.)
* Mettre à jour le contenu et les styles dynamiquement

## Technologies

* JavaScript
* HTML5
* CSS3

## Exécution

Ouvrir le fichier `.html` dans un navigateur web et interagir avec la page.
