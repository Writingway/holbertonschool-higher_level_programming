## Learning Objectives
By the end of this project, you should be able to explain to anyone, without using Google:
- How to use the Python interpreter
- How to print text and variables using `print`
- How to work with strings
- What indexing and slicing are in Python
- The official Python coding style and how to check your code with Pycodestyle

---

## Requirements
All Python scripts in this repository must follow these rules:
- **Editors allowed:** `vi`, `vim`, `emacs`
- **Interpreter:** Python 3.8.* on Ubuntu 20.04 LTS
- **File endings:** All files must end with a newline
- **Shebang:** The first line of all files must be `#!/usr/bin/python3`
- **README.md:** Mandatory at the root of the repository and the project folder
- **Pycodestyle compliance:** Version 2.7.*
- **Executability:** All files must be executable
- **File length:** Will be tested using `wc`

---

## More Information
**Pycodestyle** is the current standard for Python code style and helps maintain clean, readable code.
