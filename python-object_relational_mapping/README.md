# Python – Object Relational Mapping (ORM)

## Description

Projet Python consacré au mapping objet-relationnel (ORM), permettant d’interagir avec une base de données relationnelle en utilisant des objets Python au lieu d’écrire uniquement des requêtes SQL.

## Objectifs

- Comprendre le principe de l’ORM
- Connecter Python à une base de données MySQL
- Manipuler des tables avec des classes Python
- Utiliser une bibliothèque ORM comme SQLAlchemy

## Technologies

- Python 3
- MySQL
- SQLAlchemy

## Exécution

```bash
python3 nom_du_script.py
```
