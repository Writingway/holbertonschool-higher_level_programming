# Python – Exceptions

## Description

Projet Python dédié à la gestion des erreurs et des exceptions afin de rendre le code plus robuste.

## Objectifs

* Comprendre les exceptions
* Utiliser `try`, `except`, `else`, `finally`
* Gérer les erreurs sans interrompre le programme

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
