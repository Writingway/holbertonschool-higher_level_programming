# Python – Data Structures

## Description

Projet Python consacré aux structures de données de base : listes, tuples, chaînes et leurs manipulations.

## Objectifs

* Manipuler des listes et tuples
* Comprendre l’indexation et le slicing
* Utiliser les structures de données efficacement

## Technologies

* Python 3

## Exécution

```bash
python3 nom_du_fichier.py
```
